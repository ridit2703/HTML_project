# HTML_project


This is built with **semantic HTML** and can be easily customized to represent your own professional profile.

## Features

- Clean and structured layout with semantic HTML5
- Sections include:
  - About Me
  - Skills
  - Experience
  - Projects
  - Education
  - Contact Information
- Tables for Experience and Education
- Easily customizable content
- Fully static and lightweight

## What I learned
- Semantic HTML
- Structuring Content
- Using Headings
- Lists
- Contact Information
- Basic Styling Awareness
- Professional Structure
